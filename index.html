Hi! I'm a full-stack web developer specializing in Ruby on Rails and JavaScript. I just completed General Assembly's 12-week Web Development Immersive program. I'm now hungry for the opportunity to work for a passionate, team-oriented tech company in New York as a junior developer.

I graduated from the University of Illinois at Urbana-Champaign in Mathematics in 2006 but spent the last eight years working as a professional modern dancer in New York City, two years of which were with the historic Merce Cunningham Dance Company. Making a website that documented and shared knowledge about the craft of dance was what helped me discover my intense interest in web development and prompted me to make a career transition.

I'm especially excited by math-based code challenges and love doing Project Euler problems. Check out some of my solutions if you feel inclined. Besides honing my coding and problem-solving skills, I'm enjoying deepening my knowledge of Rails and Javascript and also exploring Backbone.js.

Projects

Conway's Game of Life

This is my favorite project that I've worked on so far. It's a simulation of John Conway's math-based, cellular automaton that allows the user to create game states, move the game forward or back in time, change grid sizes, change speeds, and load past game states. The most fun part of the project was tackling the feature of detecting oscillations and still-life objects in the game. The project was done in JavaScript and driven with Jasmine tests.

DanceShare

DanceShare was the first project I did in General Assembly's WDI and my first project in Rails. It allows audience members to write reviews of modern dance concerts. Users can create dance companies, which have runs of performances, and other users then write reviews. I still care deeply about the field of modern dance and thus about this project. The most interesting thing to me about it is that it would aggregate audience feedback across cultures and across time. I wish this had existed seventy-five years ago and had been widely used, because it would contain incredible data about the history of dance, art, and culture.
